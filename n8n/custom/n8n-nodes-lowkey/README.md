# lowkey n8n nodes

Private `n8n` nodes bundled with the `lowkey` stack:

- `VoidDB`: CRUD, query, count, collections, and blob uploads through `@voiddb/orm`
- `OpenAI / BitNet Chat`: chat completions and model listing for OpenAI-compatible APIs, including the local `BitNet` service from this repo

The `n8n` image also exposes the `openai` SDK and `@voiddb/orm` inside Code nodes.
